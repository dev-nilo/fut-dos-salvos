## üìñ About The Project

**Fut dos Salvos** is a modern web application designed to simplify the chaos of organizing recreational football matches ("peladas"). Whether you are managing a weekend squad or a competitive amateur team, this tool helps you track player presence, manage payments, and balance teams fairly based on player stats.

Built with performance and UX in mind, it leverages the power of **Next.js** and **TailwindCSS** to provide a seamless mobile-first experience.

### ‚ú® Key Features

* **üë• Squad Management:** Easily add, edit, or remove players from your roster.
* **üìä Stat Tracking:** Keep track of goals, assists, and "Man of the Match" awards.
* **‚öñÔ∏è Team Balancer:** (Optional) Algorithm to automatically generate balanced teams based on player skill levels.
* **üìÖ Match Scheduling:** Organize upcoming games and track attendance (RSVP).
* **üì± Mobile First:** Fully responsive interface for users on the go.

## üöÄ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

* **Node.js** (v18 or higher)
* **npm** or **yarn**

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/dev-nilo/fut-dos-salvos.git](https://github.com/dev-nilo/fut-dos-salvos.git)
    cd fut-dos-salvos
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Configure Environment Variables**
    Rename `.env.example` to `.env.local` and update your keys (Database URL, API Keys, etc.).
    ```bash
    cp .env.example .env.local
    ```

4.  **Run the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
